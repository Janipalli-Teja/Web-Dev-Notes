DOM

So basically Document Object Model is an API that represents and interacts with HTML or XML documents.

With HTML DOM, we can easily access and manipulate tags, IDs, classes, attributes, or elements of HTML using commands or methods provided by the document object.

Window Object

Window Object is object of the browser which is always at top of the hierarchy. It is like an API that is used to set and access all the properties and methods of the browser. It is automatically created by the browser.

The methods to access the elements in the html by javascript

  1. getElementByID()
  2. getElementByClassName()
  3. getElementByTagName()
